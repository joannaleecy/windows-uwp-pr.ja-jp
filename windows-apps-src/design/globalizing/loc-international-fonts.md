## <a name="japanese-fonts"></a>日本語フォント

Windows では Meiryo UI フォントが使用されます。 Windows には、日本語のドキュメントの表示および編集をサポートするために Yu Gothic と Yu Mincho という 2 つのフォント ファミリが付属しています。 アプリの UI に Yu Gothic はお勧めしません。
